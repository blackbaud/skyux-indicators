The key info component highlights important summary information.

Use key info to highlight important summary information.

Use key info to show the total number of items in a list. For list totals, use the horizontal key info layout and place the total at the top of the list.

Don't use key info when a different data visualization would be more appropriate. Consider using an alternative visualization such as a grid, bar chart, or line chart instead.

Don't use key info to display simple data that doesn't require extra attention or drive users to take action. Use a definition list instead.

Value Label

Use the vertical layout with the label under the primary value in most situations.

Use the horizontal layout with the label beside the primary value when displaying the total number of items in a list.

The key info component can be styled as a link in situatins where clicking on the value will navigate the user to a different location in the system.
