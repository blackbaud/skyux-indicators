Icons represent actions, content, and ideas, and they help define the visual identity of SKY UX applications. The icon component displays Font Awesome icons or custom SKY UX icons.

Icons follow key principles to deliberately facilitate user comprehension and direct focus toward actions and other content.

Icons must be well-established, universal images that visually convey meaning at a glance.

Icons are paired with action labels or other text and are not used solely for decoration or visual interest.

Icons convey a single meaning with limited variations. Each icon represents a common idea or subject that is not abstract.

Use icons alongside meaningful action labels to help users understand the purpose of actions at a glance.

Don't use icons solely for visual interest or decoration.

Don't use icons for multiple purposes. Each icon conveys a single, specific meaning that users can understand at a glance.

SKY UX uses simple, glyphic, mono-color icons to facilitate, guide, and inform users. Icons are part of Blackbaud's brand personality. They reflect a human, visionary approach and are built on the SKY UX's visual language.

Icons always use solid, monochromatic colors that they usally inherit from parent components. Icons must meet the WCAG contrast ratio of 3:1. For information about SKY UX colors, see the color guidelinescolor guidelines.

SKY UX icons have two style variations: outlined and filled. SKY UX uses outlined icons for most user experiences. It only uses filled icons to indicate state changes in components as described in the component documentation.

Icons usually inherit their size from parent elements, particularly when used inline. SKY UX icons are typically 15px16px to match standard body copy, but other sizes can be used depending on the specifications of parent components. 

SKY UX selects icons with global considerations to ensure that international users from different cultures and backgrounds can understand them.

Icons in SKY UX must meet WCAG contrast standards of a 3:1 contrast ratio with their backgrounds. 

The SKY UX icon inventory continues to grow as we approve additional use cases. To request an icon, file an issue and request attention from the SKY UX design team.
